Este projeto consiste em criar um website com NFTs
As NFTs foram criadas a partir de fotos tiradas e editadas por nós durante os 3 anos escolares. Para nós estas NFTs são simbólicas e de grande importância para o nosso coração.

O que são NFTs

Os token não-fungíveis (NFTs) são criptoativos colecionáveis exclusivos que já existem desde 2012, quando o conceito de moedas coloridas de bitcoin surgiu pela primeira vez.
Essas moedas eram simples satoshis, ou seja, pequenas frações de um bitcoin coloridas com informações distintas que poderiam conectá-las aos ativos do mundo real.
